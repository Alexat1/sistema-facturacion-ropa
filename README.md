## Descripción

Este proyecto consiste en el desarrollo de un sistema de facturación para una tienda de ropa. El sistema permitirá administrar clientes, productos y ventas, además de generar facturas y reportes.

## Funcionalidades

- Gestión de usuarios.
- Gestión de clientes.
- Gestión de productos.
- Creación de facturas.
- Cálculo automático del IVA.
- Generación de facturas en PDF.
- Reportes de ventas.
